Type: #SQL #psql #language 

- `psql --help` - info re psql
- `psql -h localhost -p5432 -U samara test` - connect to database "test" 
	- or `psql` + `\c "databasename"`
	- `\c` - connect
- `\l` - list databases
- `\d` - "describe" shows tables in database
	- `\d` *table name* - shows info on table
